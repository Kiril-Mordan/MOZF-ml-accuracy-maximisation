Five selected ml classifiers maximisation done on some financial data.

2.Logit model classifier (stats)
	- trafność: 86%
	
3.Support Vector Machines (e1071)
	- trafność: 85.67%
	
4.Naive Bayes Classifier (e1071)
	- trafność: 85%
	
5.Simple Neural Network (neuralnet)
	- trafność: 51%
